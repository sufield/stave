## Public Buckets Must Not Expose VCS Artifacts

**Control:** CTL.S3.REPO.ARTIFACT.001 (medium)

**Misconfiguration:** The bucket is publicly readable and contains version-control artifacts such as `.git/config`, `.svn/entries`, or `.env` files. These artifacts allow full repository reconstruction and can leak secrets, credentials, or internal source code.

**Raw AWS CLI commands**

```bash
aws s3api list-objects-v2 --bucket static-site --prefix .git/ --max-keys 1
aws s3api get-bucket-policy --bucket static-site
```

**Bad AWS CLI output**

VCS artifacts found in a publicly readable bucket. Save as `objects.json` and `policy.json`:

```bash
cat > objects.json <<'EOF'
{
  "Contents": [
    {
      "Key": ".git/config",
      "LastModified": "2026-03-20T14:12:01+00:00",
      "Size": 245,
      "StorageClass": "STANDARD"
    }
  ],
  "Prefix": ".git/"
}
EOF

cat > policy.json <<'EOF'
{
  "Policy": "{\"Version\":\"2012-10-17\",\"Statement\":[{\"Sid\":\"PublicReadGetObject\",\"Effect\":\"Allow\",\"Principal\":\"*\",\"Action\":\"s3:GetObject\",\"Resource\":\"arn:aws:s3:::static-site/*\"}]}"
}
EOF
```

**Extract to a Stave observation (`obs.v0.1`)**

The control checks `(properties.storage.access.public_read eq true OR properties.storage.access.public_list eq true)` AND `properties.storage.content.exposed_repo_artifacts eq true`.

```bash
mkdir -p observations

jq -n \
  --argjson objects "$(cat objects.json)" \
  --argjson policy "$(cat policy.json | jq '.Policy | fromjson')" \
'{
  schema_version: "obs.v0.1",
  generated_by: { source_type: "aws-s3-snapshot", tool: "aws-cli", tool_version: "2.x" },
  captured_at: "2026-03-20T00:00:00Z",
  assets: [{
    id: "static-site",
    type: "aws_s3_bucket",
    vendor: "aws",
    properties: {
      storage: {
        kind: "bucket",
        name: "static-site",
        access: {
          public_read: (
            [$policy.Statement[] |
              select(
                .Effect == "Allow"
                and (.Principal == "*" or .Principal.AWS == "*")
                and (.Action == "s3:GetObject" or .Action == "s3:*"
                  or ((.Action | type) == "array"
                    and (.Action | any(. == "s3:GetObject" or . == "s3:*"))))
              )
            ] | length > 0
          )
        },
        content: {
          exposed_repo_artifacts: (
            ($objects.Contents // []) | any(.Key | test("^\\.(git|svn|env)"))
          )
        }
      }
    }
  }]
}' > observations/2026-03-20T000000Z.json

jq '.captured_at = "2026-03-21T00:00:00Z"' observations/2026-03-20T000000Z.json > observations/2026-03-21T000000Z.json
```

**Detect with Stave**

```bash
stave init --profile aws-s3

stave apply \
  --observations observations \
  --max-unsafe 12h \
  --now 2026-03-21T12:00:00Z
```

**Fixed AWS CLI output**

VCS artifacts removed from the bucket. Save as `objects_fixed.json`:

```bash
cat > objects_fixed.json <<'EOF'
{
  "Prefix": ".git/"
}
EOF
```

**AWS doc basis for the fix**

Delete exposed VCS artifacts and add a deployment pipeline step that strips them before upload:

```bash
aws s3 rm s3://static-site/.git/ --recursive
aws s3 rm s3://static-site/.svn/ --recursive
aws s3 rm s3://static-site/.env
```

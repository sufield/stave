## No Public Write via ACL

ACL grants WRITE to AllUsers. Anyone on the internet can upload or delete objects in the bucket.

**Control:** CTL.S3.ACL.WRITE.001 (critical)
**Field:** `properties.storage.access.write_via_resource`

**AWS CLI command**

```bash
aws s3api get-bucket-acl --bucket upload-inbox
```

**Bad AWS CLI output**

Save as `acl.json`:

```bash
cat > acl.json <<'EOF'
{
  "Owner": {
    "DisplayName": "account-owner",
    "ID": "1111222233334444555566667777888899990000aaaabbbbccccddddeeeeffff"
  },
  "Grants": [
    {
      "Grantee": {
        "Type": "CanonicalUser",
        "ID": "1111222233334444555566667777888899990000aaaabbbbccccddddeeeeffff",
        "DisplayName": "account-owner"
      },
      "Permission": "FULL_CONTROL"
    },
    {
      "Grantee": {
        "Type": "Group",
        "URI": "http://acs.amazonaws.com/groups/global/AllUsers"
      },
      "Permission": "WRITE"
    }
  ]
}
EOF
```

**Extract to a Stave observation (`obs.v0.1`)**

```bash
mkdir -p observations

jq -n \
  --argjson acl "$(cat acl.json)" \
'{
  schema_version: "obs.v0.1",
  generated_by: { source_type: "aws-s3-snapshot", tool: "aws-cli", tool_version: "2.x" },
  captured_at: "2026-03-20T00:00:00Z",
  assets: [{
    id: "upload-inbox",
    type: "aws_s3_bucket",
    vendor: "aws",
    properties: {
      storage: {
        kind: "bucket",
        name: "upload-inbox",
        access: {
          write_via_resource: (
            [$acl.Grants[] |
              select(
                .Grantee.URI == "http://acs.amazonaws.com/groups/global/AllUsers"
                and (.Permission == "WRITE" or .Permission == "FULL_CONTROL")
              )
            ] | length > 0
          )
        }
      }
    }
  }]
}' > observations/2026-03-20T000000Z.json

jq '.captured_at = "2026-03-21T00:00:00Z"' observations/2026-03-20T000000Z.json > observations/2026-03-21T000000Z.json
```

**Detect with Stave**

```bash
stave init --profile aws-s3

stave apply \
  --observations observations \
  --max-unsafe 12h \
  --now 2026-03-21T12:00:00Z
```

**AWS doc basis for the fix**

Remove the AllUsers WRITE grant and enable Block Public Access:

```bash
aws s3api put-bucket-acl --bucket upload-inbox --acl private
aws s3api put-public-access-block --bucket upload-inbox \
  --public-access-block-configuration \
  BlockPublicAcls=true,IgnorePublicAcls=true,BlockPublicPolicy=true,RestrictPublicBuckets=true
```

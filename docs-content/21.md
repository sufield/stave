## No Public S3 Bucket Listing

**Control:** CTL.S3.PUBLIC.LIST.001 (high)

**Misconfiguration:** The bucket allows public listing through bucket policy, so object names and metadata are exposed without authentication. In S3, bucket listing is controlled by `s3:ListBucket` in policy.

**Raw AWS CLI command**

```bash
aws s3api get-bucket-policy --bucket media-assets
```

**Bad AWS CLI output**

Save as `policy.json`:

```bash
cat > policy.json <<'EOF'
{
  "Policy": "{\"Version\":\"2012-10-17\",\"Statement\":[{\"Sid\":\"PublicListBucket\",\"Effect\":\"Allow\",\"Principal\":\"*\",\"Action\":\"s3:ListBucket\",\"Resource\":\"arn:aws:s3:::media-assets\"}]}"
}
EOF
```

**Extract to a Stave observation (`obs.v0.1`)**

The control checks `properties.storage.access.public_list`. The extractor parses the bucket policy and checks whether any `Allow` statement grants `s3:ListBucket` to `Principal: "*"`.

```bash
mkdir -p observations

cat policy.json | jq '
  (.Policy | fromjson) as $p |
{
  schema_version: "obs.v0.1",
  generated_by: { source_type: "aws-s3-snapshot", tool: "aws-cli", tool_version: "2.x" },
  captured_at: "2026-03-20T00:00:00Z",
  assets: [{
    id: "media-assets",
    type: "aws_s3_bucket",
    vendor: "aws",
    properties: {
      storage: {
        kind: "bucket",
        name: "media-assets",
        access: {
          public_list: (
            [$p.Statement[] |
              select(
                .Effect == "Allow"
                and (.Principal == "*" or .Principal.AWS == "*")
                and (
                  .Action == "s3:ListBucket" or .Action == "s3:*"
                  or ((.Action | type) == "array"
                    and (.Action | any(. == "s3:ListBucket" or . == "s3:*")))
                )
              )
            ] | length > 0
          )
        }
      }
    }

}' > observations/2026-03-20T000000Z.json
```

**Fixed AWS CLI output**

The public `ListBucket` statement is replaced with a specific principal. Save as `policy_fixed.json`:

```bash
cat > policy_fixed.json <<'EOF'
{
  "Policy": "{\"Version\":\"2012-10-17\",\"Statement\":[{\"Sid\":\"AllowAppReadOnly\",\"Effect\":\"Allow\",\"Principal\":{\"AWS\":\"arn:aws:iam::123456789012:role/AppReadRole\"},\"Action\":\"s3:GetObject\",\"Resource\":\"arn:aws:s3:::media-assets/*\"}]}"
}
EOF
```

**AWS doc basis for the fix**

Remove public `s3:ListBucket` policy grants. Replace with specific principals if listing access is needed. Enable S3 Public Access Block to prevent future public grants.

## Transport Encryption Required

**Control:** CTL.S3.ENCRYPT.002 (high)

**Misconfiguration:** The bucket policy does not deny non-HTTPS requests, so data could be sent over plaintext HTTP instead of HTTPS. AWS recommends enforcing encrypted transport with the `aws:SecureTransport` condition key in the bucket policy.

**Raw AWS CLI command**

```bash
aws s3api get-bucket-policy --bucket reports
```

**Bad AWS CLI output**

The bucket policy allows read access but has no deny statement for insecure transport. Save as `policy.json`:

```bash
cat > policy.json <<'EOF'
{
  "Policy": "{\"Version\":\"2012-10-17\",\"Statement\":[{\"Sid\":\"AllowAppRead\",\"Effect\":\"Allow\",\"Principal\":{\"AWS\":\"arn:aws:iam::123456789012:role/AppReadRole\"},\"Action\":\"s3:GetObject\",\"Resource\":\"arn:aws:s3:::reports/*\"}]}"
}
EOF
```

**Extract to a Stave observation (`obs.v0.1`)**

The control checks `properties.storage.encryption.in_transit_enforced`. The extractor parses the bucket policy and looks for the standard AWS deny pattern: a `Deny` statement with `aws:SecureTransport` set to `"false"`.

```bash
mkdir -p observations

cat policy.json | jq '
  (.Policy | fromjson) as $p |
{
  schema_version: "obs.v0.1",
  generated_by: { source_type: "aws-s3-snapshot", tool: "aws-cli", tool_version: "2.x" },
  captured_at: "2026-03-20T00:00:00Z",
  assets: [{
    id: "reports",
    type: "aws_s3_bucket",
    vendor: "aws",
    properties: {
      storage: {
        kind: "bucket",
        name: "reports",
        encryption: {
          in_transit_enforced: (
            [$p.Statement[] |
              select(
                .Effect == "Deny"
                and (.Condition.Bool["aws:SecureTransport"] == "false"
                  or .Condition.Bool["aws:SecureTransport"] == false)
              )
            ] | length > 0
          )
        }
      }
    }
  }]
}' > observations/2026-03-20T000000Z.json
```

**Fixed AWS CLI output**

A `Deny` statement with `aws:SecureTransport: "false"` has been added. Save as `policy_fixed.json`:

```bash
cat > policy_fixed.json <<'EOF'
```

**AWS doc basis for the fix**

Add this statement to the bucket policy:

```json
{
  "Sid": "DenyInsecureTransport",
  "Effect": "Deny",
  "Principal": "*",
  "Action": "s3:*",
  "Resource": [
    "arn:aws:s3:::reports",
    "arn:aws:s3:::reports/*"
  ],
  "Condition": {
    "Bool": {
      "aws:SecureTransport": "false"
    }
  }
}
```

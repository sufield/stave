Offline Security CLI

Configuration safety, simplified.

Stave helps you validate, plan, apply, and diagnose infrastructure risk from snapshots. No cloud credentials, no remote calls, and deterministic outputs for repeatable CI runs.

A clear operational workflow

Move from input checks to evaluated findings and targeted remediation without guesswork.

1. Validate and plan Catch schema and readiness issues before running the evaluator. stave validate --controls ./controls --observations ./obs stave plan --controls ./controls --observations ./obs
2. Apply controls Evaluate snapshots and produce machine-readable findings. stave apply --controls ./controls --observations ./obs \ --max-unsafe 7d --format json > evaluation.json
3. Diagnose and verify Explain findings, track fixes, and verify remediation impact. stave diagnose --previous-output evaluation.json stave verify --before ./obs-before --after ./obs-after

cd stave
make build
./stave quickstart
